Cache file tabulator
This page can be used to create a table of all items with specific fields available in the cache. Select the cache type, then input at least one search arg. Every entity with at least one of the search args present will be returned. You can additionally add some view args, which will be displayed for the returned entities, but not used for searching. By default 'id' and 'name' and always set as view args.
You can ask for contents of object or array fields by using dot notation - for example, to get the keepsake_override parameter of extra, use extra.keepsake_override. Example: try extra.extra_0011F2 as a search arg and view args equipId and equipSlotId.
If you want to view the output with full filtering/sorting/etc., I encourage you to download the tsv file using the link then import that into a spreadsheet program like Excel or Google Sheets. If you accidentally close/refresh the page without downloading the file, feel free to let me (Gaz) know and I can likely fetch it from the server for you.
